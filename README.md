# Ocean-Pong

A modern Pong clone built in C++ using SFML.

## Features
- Single-player AI
- Two-player mode
- State machine
- Randomized serves
- Game Over screen
- Score tracking

## Controls
### Player 1(left)
- W - Up
- S - Down
### Player 2(right)
- ↑ - Up
- ↓ - Down
